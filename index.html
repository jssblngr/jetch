This Etch-a-sketch is an individual project for The Odin Project's Foundations Cirriculum.

This project is designed to work on a desktop browser. Select a color option and then hover your cursor over the canvas to draw. Due to using the cursor, this will not function on a mobile device.

"Adjust DPI" will change how many "pixels" are inside the etch-a-sketch. Clearing will clear you canvas. You can choose between Black, Greyscale (which also allows shading), and a random color generator called Rainbow. You can also select a custom color using an HTML5 RGB selector.

Currently, changing your color option will reset the canvas. This issue will be addressed in a future update.

This project was created by Jesse Belanger using vanilla Javascript, CSS, and HTML.
